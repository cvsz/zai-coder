# Production Handoff

## What this package is

A complete source scaffold and local-first control plane foundation.

## What this package is not yet

A fully hardened internet-facing SaaS.

## Before internet exposure

- Enforce API auth middleware.
- Add session auth or SSO.
- Use HTTPS and Cloudflare Access.
- Move secrets to environment/secret manager.
- Run secret scan.
- Run dependency scan.
- Add monitoring.
- Add backups.
- Review billing provider compliance.
