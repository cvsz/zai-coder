# Archived Build Reports

This directory contains legacy build report snapshots that were previously stored at the repository root.

These files are preserved for historical and audit reference only. They are not runtime source files, package assets, release artifacts, or generated files required by the active release workflow.

## Archive Policy

- Build reports should not be committed at the repository root.
- Historical reports are kept under `docs/archive/build-reports/` for traceability.
- Future generated build reports should be treated as local/generated output unless explicitly promoted to documentation.
- Release artifacts must continue to live outside git-tracked `dist/` outputs unless intentionally documented through release notes or verification records.

## Current Context

This archive was created after the v0.1.3 release workflow completed. The cleanup is documentation-only and does not change package source, release tags, or GitHub release assets.
