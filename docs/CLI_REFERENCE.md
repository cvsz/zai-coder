# CLI Reference
Base Command: `./zai-coder`

- `self heal --check`: Parse failures, generate repair plans
- `repair check <patch>`: Validate local fix paths 
- `index build`: Scan local directories natively
- `rag query <term>`: Pull context subsets
- `task list`: Active local queue elements
- `eval run`: Invoke system-wide mock benchmark regressions
- `metrics snapshot --json`: Output monitoring state
- `deploy plan --target <sys>`: Generate local deployment schemas
- `tui --dry-run`: Invoke visual dashboard
