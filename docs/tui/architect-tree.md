# Architect Tree

`tui-template-04` / `architect-tree` maps local system relationships.

## Layout

- Left tree navigation for Agents, Skills, Commands, Config, Docs, Tests, and Release.
- Main detail panel for the selected node.
- Bottom status/action panel for local dry-run actions.

## Keyboard

- `q`: quit
- Arrow keys: navigate tree
- `enter`: open node
- `ctrl+k`: command palette
- `ctrl+r`: refresh
- `f1`: help
