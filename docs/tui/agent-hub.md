# Agent Hub

`tui-template-02` / `agent-hub` is the operator dashboard for multiple local agents.

## Layout

- Header with local operations status.
- Tile grid for Core Agent, Release Agent, Safety Agent, Install Agent, TUI Agent, and Memory Agent.
- Right command panel for safe local actions.
- Bottom log panel with redacted local output.

## Keyboard

- `q`: quit
- `tab`: cycle focus
- `ctrl+k`: command palette
- `ctrl+r`: refresh
- `f1`: help
