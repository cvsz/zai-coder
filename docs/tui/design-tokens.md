# TUI Design Tokens

Terminal UIs cannot implement browser backdrop blur. ZAI Coder uses a terminal-native glassmorphism-inspired style instead:

- Layered panels
- Soft rounded borders
- Dim and bright contrast
- Intentional spacing
- Status chips
- Command palette
- Non-blocking async refresh
