# Creative Canvas

`tui-template-05` / `creative-canvas` is the preview-heavy workspace for creative, docs, prompts, screenshots, and generated artifacts.

## Layout

- Left prompt/session panel.
- Center preview canvas with text preview, file preview placeholder, and media placeholder.
- Right metadata/status panel.
- Bottom command input.

## Keyboard

- `q`: quit
- `ctrl+p`: preview panel focus
- `ctrl+k`: command palette
- `ctrl+r`: refresh
- `f1`: help
