# Command Center

`tui-template-01` / `command-center` is the focused deep-work template.

## Layout

- Header with project, provider/model, template, and safe mode.
- Main chat/work panel for operator notes and command output.
- Right status sidebar with workspace, dry-run mode, session, selected template, and last command.
- Bottom command input.
- Floating command palette via `ctrl+k`.

## Keyboard

- `q`: quit
- `ctrl+k`: command palette
- `ctrl+r`: refresh
- `ctrl+d`: toggle dry-run mode
- `f1`: help
