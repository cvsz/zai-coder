# Flow Stream

`tui-template-03` / `flow-stream` shows sessions, agent events, commands, and outputs in chronological order.

## Layout

- Vertical timeline.
- Event cards with timestamp, actor, action, status, and summary.
- Right filter panel: all, commands, agents, safety, errors.
- Bottom command/log input.

## Keyboard

- `q`: quit
- `j/k`: move through timeline
- `ctrl+f`: filter
- `ctrl+r`: refresh
- `f1`: help
