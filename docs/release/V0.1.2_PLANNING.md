## Known Warning Baseline (v0.1.1)
**Source**: `python3 -m pytest -q` on `f3eff06`

### Warning 1: PytestCollectionWarning
```
zai_coder/quality_assurance_test_lab/models.py:9
  PytestCollectionWarning: cannot collect test class 'TestCase' because it has
  a __init__ constructor (from: tests/test_quality_assurance_test_lab_v43.py)
```
**Root cause**: `TestCase` dataclass uses `@dataclass(frozen=True)` with an `__init__` that confuses pytest's collection heuristic.
**Fix target**: Rename the class or add `pytest.ini` option `python_classes = Test` exclusion.

### Warning 2: PytestReturnNotNoneWarning
```
tests/test_quality_assurance_test_lab_v43.py::test_matrix
  PytestReturnNotNoneWarning: Expected None, but test_matrix returned a list
```
**Root cause**: `test_matrix` returns a list instead of using `assert`.
**Fix target**: Replace `return` with `assert` or store result in a local variable.
