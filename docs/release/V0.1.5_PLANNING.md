# ZAI Coder v0.1.5 Planning

## Strategic Phase: Intelligent Media & Advanced Modalities

With the core agent runner orchestration and safe local environment stabilized in v0.1.4, ZAI Coder v0.1.5 pivots to integrating richer data types and expanding the cognitive context beyond raw text.

### High-Level Objectives

1. **Media Browser Ecosystem**: Introduce modular plugins for local image rendering, basic audio playback, and video frame extraction inside the TUI context.
2. **Vision/Voice Toolsets**: Stand up the core abstractions for visual understanding (passing images to VLM providers) and Voice API connectivity.
3. **Enhanced MCP Implementations**: Expand upon the v0.1.4 provider routing to include real-world robust MCP servers (e.g., local database exploration, extended git integration).
4. **Resiliency and Metrics**: Introduce telemetry (opt-in local sqlite logs) to track agent reasoning loop times, token usage, and retry saturation.

### Release Boundaries
- Does **not** include cloud deployment or remote daemons (remains strictly local).
- Version stays at `0.1.4` across all codebase files until the release candidate phase.
