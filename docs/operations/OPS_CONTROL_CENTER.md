# Operations Control Center

## Purpose

A local-first operations panel for deployment status, health, logs, backup, upgrade, rollback, and service actions.

## Commands

```bash
make ops-control-center
make ops-status-panel
make ops-health-dashboard
make ops-log-viewer
make ops-backup-dashboard
make ops-upgrade-dashboard
make ops-action-plans
```

## Local routes

```text
/api/ops/status
/ops
/ops/services
/ops/health
/ops/backup
/ops/upgrade
```
