# Developer Portal and API Docs Report

## Endpoints

- GET /api/status: Read local control-plane status.
- GET /api/team/status: Read team collaboration status.
- GET /api/notifications/status: Read notification center status.
- GET /api/help/status: Read help center status.
- GET /api/content-studio/status: Read template content studio status.
